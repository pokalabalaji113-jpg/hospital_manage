# 🏥 Full Stack Hospital Management System

## 📌 Project Overview
The Hospital Management System is a full-stack web application designed to manage hospital operations efficiently.  
It provides functionality for managing patients, doctors, appointments, and medical records.

This project demonstrates full-stack development using modern backend, frontend, and database technologies.

---

## 🧱 Tech Stack

### 🔹 Frontend
- Streamlit

### 🔹 Backend
- FastAPI

### 🔹 Database
- PostgreSQL

### 🔹 Version Control
- Git & GitHub

---

## 🚀 Key Features

- 📝 Patient Registration & Management  
- 👨‍⚕️ Doctor Management  
- 📅 Appointment Scheduling  
- 🗂 Medical Record Storage  
- 🔐 Backend API Integration  
- 📊 Interactive UI with Streamlit
