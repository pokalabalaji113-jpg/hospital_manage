# 🏥 Full Stack Hospital Management System

## 📌 Project Description
This is a Full Stack Hospital Management System built using Python.  
The application allows management of patients, doctors, and appointments through a structured backend API and interactive frontend interface.

---

## 🧱 Tech Stack

### 🔹 Backend
- FastAPI

### 🔹 Frontend
- Streamlit

### 🔹 Database
- PostgreSQL

### 🔹 Version Control
- Git & GitHub

---

## 🚀 Features

- Patient Management  
- Doctor Management  
- Appointment Booking  
- Database Storage using PostgreSQL  
- REST API integration with FastAPI  
- Interactive UI with Streamlit  

---
